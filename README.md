## 简介

- 部署环境:Centos 7.3(两台)
- 架构 controller + compute
- Core services:
	+ Keystone
	+ Nova
	+ Glance
	+ Neutron
	+ Swift (in test)
	+ Cinder (controller only)
	+ Horizon
